# 🎯 KANJI SNIPER: SCREEN OVERLAY OCR

![Version](https://img.shields.io/badge/Version-4.0.0--LITE-red)
![Engine](https://img.shields.io/badge/Engine-Tesseract.js-blue)
![Stack](https://img.shields.io/badge/Stack-WebRTC_%2F_Canvas-emerald)

**Kanji Sniper** is a high-performance screen overlay utility for real-time Japanese text recognition. Designed for language learners and researchers, it allows you to "snipe" and extract kanji directly from your screen using a live video feed or screen share.

## ✨ Features

- 🎯 **Precision Sniping**: Highlight specific areas of your screen to trigger instant OCR.
- 📡 **WebRTC Integration**: Seamlessly captures live video feeds or browser tabs for scanning.
- 🧠 **Tesseract Core**: Powered by **Tesseract.js** for robust, client-side optical character recognition.
- 🎨 **Dynamic Overlay**: A lightweight HTML5 Canvas layer that provides non-intrusive UI feedback.

## 🚀 Technical Stack

- **OCR Engine**: Tesseract.js
- **Vision**: WebRTC MediaDevices API
- **UI/UX**: HTML5 Canvas / Vanilla JavaScript
- **v4.0 Lite**: Optimized for low-latency performance on standard hardware.

## 🛠️ Usage

1. Open the application in a modern browser.
2. Grant permission for screen/tab capture.
3. Use the cursor to "snipe" (drag and select) the kanji text you wish to recognize.
4. The extracted text will appear in the UI for instant copying or look-up.
